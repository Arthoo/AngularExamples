Generic lists in Angular

In this example you will see how we can create a list component where we can specify which component to use for displaying the items. This can be very useful if you would like to have several lists in your application with the same style and functionalities for search/filter/pagination but with different content.

The two lists below are instances of the same list component but uses different components to display the items inside.

Let's see how this can be done with the following example. In our list component we have an input for the list of item data, and a template reference with the component to be used.

Inside the list template we iterate over the list items and use *ngTemplateOutlet to instantiate the referenced component for each item. By setting the item data in the context we can provide it to the component we instantiate.

When using the list component, we can simply provide the item component in a template inside. The item data can be accessed by assigning a template input variable (let-cat="item" in this example).
